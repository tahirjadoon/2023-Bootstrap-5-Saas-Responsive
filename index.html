Containers

Containers worksheet Bootstrap Containers
  • .container: which sets a max-width at each responsive breakpoint
  • .container-fluid: which is width: 100% at all breakpoints
  • .container-{breakpoint}: which is width: 100% until the specified breakpoint
breakpoint = -sm, -md, -lg, -xl, -xxl

h & p
  • p and h use font-size: calc() for xs screen to make sure it is responsive
  • h1-h6: margin-bottom .5rem
  • headers can also be applied as class to p tag like <p class="h1">Something</p>
  • Can apply text-muted, display-# (1-6)
  • p: margin-bottom 1rem
  • Can add classes to p like lead
  • Other tags are mark, del, s, ins, small, strong, em, abbrev with title, blockquote, figure, figcaption with cite
  • classes text-center, text-end, text-start
  • ul: list-unstyled, list-inline, list-inline-item
lists
  • ul, ol, dl: margin-bottom 1rem
  • ul, ol: padding-left has been reset
  • nested lists have no margin-bottom
  • dd: marging-bottom .5rem
  • dt: are bolded
