Basic Form

  • Form has no class applied
  • Text has class form-text applied
  • Label has class form-label applied
  • Input has class form-control applied
  • Checkbox has class form-check-label applied and the label for the check box has form-check-input applied

Form with Gutters, Alignment and Aria Describedby

  • Form has classes row g-3 align-items-center border border-2 border-danger rounded applied
  • Each div has classes col-auto bg-{primary/success/warning} and background applied
  • Label has class col-form-label applied
  • Textbox has class form-control applied
  • Textbox also has aria-describedby applied
  • Descriptive text has class form-text applied
Must be 8 to 20 chars long

Disabled Form Control

  • Form has no class applied
  • Text has class form-text applied
  • Label has class form-label applied
  • Input has class form-control applied
  • Checkbox has class form-check-label applied and the label for the check box has form-check-input applied
  • Email textbox has disabled atribute applied to it
