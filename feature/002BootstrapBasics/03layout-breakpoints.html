Breakpoints

Breakpoint Class infix Dimensions
X-Small None <576px [0 to 575.98px]
Small sm ≥576px [576px to 767.98px]
Medium md ≥768px [768px to 991.98px]
Large lg ≥992px [992px to 1199.98px]

Header 1 for response design

Css for above example

/* DEFAULT: keep it before others, greater than x-large -- covering xx-large */
.responsive{background-color: lightblue;}
.header {background-color: gray; color: lightpink;}

/* upto x-small */
@media only screen and (max-width: 575.98px){
 .responsive {background-color: blue;}
 .header {background-color: black; color: red;}
}

/* greater than x-small but upto small */
@media only screen and (min-width: 556px) and (max-width: 767.98px){
 .responsive {background-color: bisque;}
 .header {background-color: brown; color: white;}
}

/* greater than small but upto medium */
@media only screen and (min-width: 768px) and (max-width: 991.98px){
 .responsive {background-color: violet;}
 .header {background-color: tomato; color: yellowgreen;}
}

/* greater than medium but upto large */
@media only screen and (min-width: 992px) and (max-width: 1199.98px){
 .responsive {background-color: saddlebrown;}
 .header {background-color: violet; color: blueviolet;}
}

/* greater than large but upto x-large */
@media only screen and (min-width: 1200px) and (max-width: 1399.98px){
 .responsive {background-color: red;}
 .header {background-color: tan; color: teal;}
}
